# Hide and Seek in City - Data Pack
Minecraft 1.21.11

## Quick Start
```
/function has:play   — Start the game
/function has:reset  — Reset all values
/function has:help   — Show all settings and commands
```

## Settings
Use `/scoreboard players set <score> has_setting <value>` to configure:

| Setting    | Values | Default | Description              |
|------------|--------|---------|--------------------------|
| `border`   | 1/2/3  | 1       | 100 / 150 / 200 blocks   |
| `timer`    | 1/2/3  | 1       | 5 / 10 / 15 minutes      |
| `countdown`| 1/2/3  | 1       | 1 / 2 / 3 min seeker lockout |
| `shrink`   | 0/1    | 0       | Shrink border at half-time |
